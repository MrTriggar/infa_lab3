# Учет заказов в интернет-магазине

## О проекте

Проект представляет собой сервис для учета заказов в интернет-магазине. Сервис собирает 
информацию о покупателях, продавцах, товарах и заказах. Сервис воссоздает типичный функционал 
интернет-магазина, в который входит:
- Поиск товаров
- добавление товаров в корзину
- отбор товаров для составления заказа
- создание заказа

Для функционала в дальнейшем будут реализованы API и прочие функции.

## Принцип работы сервиса

Для хранения данных, в коде программы должны быть реализованы модели таблиц. Для этого будет использована
библиотека `sqlalchemy`. Модели таблиц будут храниться в файле `src/main/database.py`.

Модель таблицы будет иметь следующую структуру:
```python
class Test(Base):
    __tablename__ = "test_table"
    
    id: Mapped[int] = mapped_column(primary_key=True)
    some_column_1: Mapped[str]
    some_column_2: Mapped[str]
```
Класс представляет из себя таблицу. 
Поля класса представляют из себя колонки таблицы. Поле `__tablename__` задает имя таблицы.

В таблицах будут сохранены данные:
1. О заказе:
    - количество товаров, собранных в заказе
    - цена
    - место выдачи
    - дата выдачи
    - уникальный номер
    - способ доставки

2. О товаре:
    - цена
    - категория товара
    - статус товара(в наличии или нет)
    - продавец
    - оценка товара

3. О покупателе:
    - Имя Фамилия
    - Почта
    - Информация о прошлых покупках
    - Оцененные товары



## Что нужно для запуска?

- Git
- Python 3.13
- docker и docker compose

## Запуск проекта

1. Клонирование репозитория с github

```git clone https://github.com/MrTriggar/infa_lab3.git```

2. Переход в корень проекта

```cd ```

3. Создание виртуального окружения

```python -m venv venv```

4. Активация виртуального окружения

```venv\Scripts\activate```

5. Установка зависимостей

```pip install -r requirements.txt```

6. Запуск работы

```python scr/main/main.py```
